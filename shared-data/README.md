# Shared Data

Reusable, invented e-commerce sample data shared across use cases lives here.

The dataset (a small product catalog, campaigns, customers, and orders) will be generated from a **deterministic seed script** before use case #1, so every demo draws from the same consistent, reproducible sample. Reuse this data instead of inventing new datasets per use case.

All data here is fictional. It contains no real customers, orders, brands, or personal information.

> Not generated yet — the seed script and generated files are added with use case #1.
