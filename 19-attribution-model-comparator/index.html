Which channel gets the credit? It depends on the model.

Attribution is a choice, not a truth. This rebuilds each conversion's channel journey for the fictional Northstar Outfitters store and scores it five ways — first-click, last-click, linear, position-based, and time-decay. The same revenue, redistributed: last-click flatters the closer, first-click flatters the introducer. Watch the credit move, measure the last-click bias channel by channel, and open any journey to see exactly how each model splits it. Deterministic, client-side — the model is a decision you should make on purpose, not an invisible default.
